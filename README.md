# employee-management

## Getting Started

### 1. Build the Project

```bash
mvn clean compile
```

### 2. Run the Application

#### Using Maven:

```bash
mvn spring-boot:run
```

### 3. Test the Application

The application will start on `http://localhost:8080`

#### Available Endpoints:

- **GET** `/` - Welcome message
- **GET** `/health` - Health check
